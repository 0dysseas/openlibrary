Coverthing

API:

/api/get/id : return all properties of thing with that id.
example: /api/get/1

/api/query: returns ids of all things that match the query.
example: /api/query?isbn=0000000183

/api/upload: Please don't use this now. This is not stable yet.

IMAGES:

/key/value/size.jpg - gives the image of the thing having property key with specified value with in the specified size. Supported sizes are small 100x100, medium 200x200 and large 400x400.
examples:

  • /isbn/0000000183/small.jpg
  • /id/1/small.jpg
